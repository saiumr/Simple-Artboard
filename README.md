### 实现：C + SDL2  
若承蒙您的错爱还请自行安装SDL2库之后再使用  
  
### 运行(Ubuntu MacOS Linux)  
``` 
$./run.sh
```  
  
### 画板粒度  
默认20：  
![preview1](https://github.com/saiumr/saiumrImages/blob/master/Simple-Artboard1.gif)
